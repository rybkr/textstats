# textstats

`textstats` is a small, self-contained Python library for computing basic
statistics on text with the Python standard library only.

## Features

- `word_count(text)`
- `sentence_count(text)`
- `average_word_length(text)`
- `most_common_words(text, n=5)`
- `reading_time(text, wpm=200)`
- `lexical_diversity(text)`

## Development

Install the package and test dependencies, then run:

```bash
pytest
```
